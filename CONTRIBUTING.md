# Contributing to DH_Aspect Lock

Thank you for considering contributing to this Blender add-on! Your feedback, bug reports, and pull requests are welcome.

## How to Contribute
- **Bug Reports:** Open a GitHub Issue with detailed steps to reproduce the problem, Blender version, and screenshots if possible.
- **Feature Requests:** Open an Issue and describe your requested feature and its intended use-case.
- **Code Contributions:**
  1. Fork this repository
  2. Create a new branch for your fix/feature
  3. Ensure your code follows Blender's Python add-on guidelines
  4. Test with recent Blender versions (3.x, 4.x, 5.x if possible)
  5. Write clear commit messages
  6. Submit a Pull Request with a description of your changes

## Code Style
- Follow PEP8 for Python scripts
- Use descriptive function and variable names
- Add comments and docstrings where needed

## Tests
- If possible, provide steps or .blend files to demonstrate/test bugs or features.

## Other Notes
- Please respect the [Code of Conduct](CODE_OF_CONDUCT.md)
- For major changes, open an Issue to discuss before you start
