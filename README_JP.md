# DH_Aspect Lock (Auto Aspect Lock / 解像度比率固定)

Blender用アドオン「Auto Aspect Lock」は、出力プロパティの解像度XまたはYを変更した際に、アスペクト比を自動維持するツールです。

---

## 主な機能

* 解像度X/Yのアスペクト比を自動固定
* XかYどちらか一方を変更するだけで、もう一方が自動追従
* 出力プロパティのパネルからON/OFF切り替え可能
* Blender標準のFormat解像度UIと統合
* 軽量、外部依存なし

---

## 対応バージョン

* Blender 3.0 以降
* Blender 4.x / 5.x 動作想定

---

## インストール方法

1. `DH_Aspect Lock.py` を保存
2. Blenderを開く
3. `編集 > プリファレンス > アドオン`
4. 「インストール」を押す
5. `DH_Aspect Lock.py` を選択
6. チェックをON

---

## 使用方法

1. `出力プロパティ` → `フォーマット` を開く
2. `解像度比率を固定` にチェックを入れる
3. `解像度X` または `解像度Y` を変更
4. もう一方の値が自動的に追従します

### 注意
- ロックON時のアスペクト比が基準として記憶されます
- ロックOFF時は通常通り自由に編集できます

---

## UI場所

```
出力プロパティ
 └ フォーマット
    └ 解像度比率を固定 (チェックボックス)
```
