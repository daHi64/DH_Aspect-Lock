# Security Policy for DH_Aspect Lock

## Supported Versions
- Latest release

## Reporting a Vulnerability
If you find a vulnerability or security issue, please **do not open a public issue**. Instead, contact the maintainer via email:

- dahi64@users.noreply.github.com

You can also use GitHub's "Report a vulnerability" function in the repository's Security tab.

Upon receiving your report, we will investigate and respond as soon as possible.
